# JWT-sample backend (원티드 프리온보딩코스)

## Description

[NestJS](https://github.com/nestjs/nest) 를 이용해 구축한 아주 기본적인 JWT 인증을 지원하는 백엔드 어플리케이션입니다.


## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
